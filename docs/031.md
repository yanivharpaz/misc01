Here's how to create a symmetric circular badge that adapts to 1-3 digits:

```html
<!DOCTYPE html>
<html>
<head>
<style>
.badge {
  display: inline-flex;
  align-items: center;
  justify-content: center;
  min-width: 24px;
  min-height: 24px;
  padding: 4px;
  background-color: #e53e3e; /* red */
  color: white;
  border-radius: 50%;
  font-size: 12px;
  font-weight: bold;
  font-family: Arial, sans-serif;
  aspect-ratio: 1 / 1; /* keeps it square/circular */
}
</style>
</head>
<body>
  <span class="badge">5</span>
  <span class="badge">42</span>
  <span class="badge">999</span>
</body>
</html>
```

**Key techniques:**

1. **`aspect-ratio: 1 / 1`** - Ensures the badge stays square (which becomes circular with border-radius: 50%)
2. **`min-width` and `min-height`** - Sets a minimum size for single digits
3. **`display: inline-flex`** with **`align-items` and `justify-content: center`** - Centers the number perfectly
4. **`border-radius: 50%`** - Makes it circular
5. **Equal padding** - Maintains symmetry as content grows

**Alternative approach** (if you want a truly fixed circle size and to abbreviate large numbers):

```css
.badge {
  display: inline-flex;
  align-items: center;
  justify-content: center;
  width: 20px;
  height: 20px;
  background-color: #e53e3e;
  color: white;
  border-radius: 50%;
  font-size: 11px;
  font-weight: bold;
}
```

Then use JavaScript to show "99+" for numbers over 99 to keep it always circular.
